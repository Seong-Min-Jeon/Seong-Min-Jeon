## 👋 About Me

안녕하세요, **전성민(Seongmin Jeon)**입니다.  
저는 **문서 기반 AI 서비스, 검색·QA 시스템, 데이터 분석 자동화, LLM 백엔드 설계**에 관심이 많은 개발자입니다.

단순히 모델을 사용하는 데서 끝나지 않고,

- 검색 품질을 높이기 위한 구조를 설계하고
- LangGraph 기반의 흐름 제어와 에이전트 오케스트레이션을 구성하고
- VectorDB 기반 검색 및 분석 시스템을 만들고
- Docker / Linux 환경에서 실제 서비스와 학습 환경을 운영하고
- 반복 가능한 형태의 AI 시스템으로 정리하는 일

을 하고 있습니다.
