안녕하세요, **전성민(Seongmin Jeon)**입니다.

현재는 공공·행정·연구보고서와 같은 도메인 문서를 다루는 LLM 서비스 개발을 경험하고 있으며,  
Python을 중심으로 실제 동작하는 AI 시스템을 만드는 데 집중하고 있습니다.

단순히 LLM을 호출하는 데서 끝나지 않고,

- 문서를 검색 가능한 형태로 전처리하고
- RAG 기반 질의응답 흐름을 구성하고
- 자연어 질문을 SQL 질의로 연결하고
- LLM 응답의 근거성과 정확성을 검토하고
- Docker / Linux 환경에서 서비스를 실행하고
- 반복 가능한 형태의 AI 시스템으로 정리하는 일

에 관심이 많습니다.

## 🧩 경험과 관심사

### LLM 서비스 개발

- RAG 기반 문서 검색 및 질의응답 흐름 구성
- 연구보고서, 행정 문서, 공공 도메인 문서 기반 AI 서비스 개발 경험
- 검색 결과를 바탕으로 LLM 답변을 생성하는 구조 설계
- 답변이 실제 근거 문서에 기반하는지 검토하는 평가 경험
- 프론티어 모델을 활용한 답변 정확도 및 근거성 검증 경험

### Text-to-SQL / SQL Agent

- 자연어 질문을 SQL 질의로 변환하는 시스템 개발 경험
- 읽기 전용 SQL 생성 및 실행 전 검증 흐름 구성
- SQL Agent의 잘못된 컬럼 매핑을 줄이기 위한 전용 View 설계
- 세션 기반 대화 이력 저장을 통한 멀티턴 질의 처리 경험

### Local LLM / 도메인 지식 활용

- Open-weight LLM을 활용한 도메인 특화 AI 시스템에 관심
- 보안·제약 환경에서 활용 가능한 로컬 LLM 구조에 관심
- RAG, 파인튜닝, 지식 주입, 모델 평가 방법에 관심
- 프론티어 모델과 로컬 모델의 역할 분담 방식에 관심

### 교육과 커뮤니케이션

- 기술 개념을 쉽게 설명하고 구조화하는 일에 관심
- LLM, Embedding, RAG, Text-to-SQL 관련 내부 학습 자료 작성 경험
- AI 기초, 통계, 딥러닝 개념을 장기간 커리큘럼 형태로 설명한 경험
- 개발자와 비개발자 사이에서 기술 내용을 풀어 설명하는 데 강점
